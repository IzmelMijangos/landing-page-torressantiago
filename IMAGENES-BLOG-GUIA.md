# Guía de Descarga de Imágenes para Blog - Torres Santiago

Esta guía te ayudará a descargar imágenes profesionales y gratuitas de Unsplash para tu blog.

## Instrucciones Generales

1. Haz clic en el enlace de búsqueda de Unsplash
2. Selecciona una imagen que te guste (preferiblemente horizontal/landscape)
3. Descarga en alta resolución (botón "Download free" o similar)
4. Renombra el archivo con el nombre exacto indicado
5. Coloca el archivo en `/public/images/blog/`

**Tamaño recomendado:** 1200x630px (ideal para SEO y OpenGraph)

## Próximos Pasos

Una vez que hayas descargado las imágenes:

1. Colócalas todas en `/public/images/blog/`
2. Verifica que los nombres coincidan exactamente (incluyendo extensión .jpg)
3. Avísame y te ayudaré a optimizarlas para web (compresión, tamaño correcto)

## Tips para Seleccionar Imágenes

✅ **Prefiere:**
- Imágenes horizontales (landscape)
- Alta resolución
- Colores vibrantes pero profesionales
- Composición limpia

❌ **Evita:**
- Imágenes con texto incrustado
- Demasiado oscuras o saturadas
- Con marcas de agua
- Verticales (portrait)

## Alternativa Rápida

Si prefieres que te ayude con placeholders temporales mientras consigues las imágenes finales, también puedo implementar eso. Solo avísame.

---

**Nota:** Todas las imágenes de Unsplash son gratuitas para uso comercial. No necesitas atribución, pero es una buena práctica mencionar al fotógrafo en tu aviso de privacidad.
